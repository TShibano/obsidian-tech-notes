## 概要

エージェントスキル（Agent Skills）とは、AI コーディングアシスタントや自律エージェントに対して再利用可能な命令セットを定義する仕組みである。`SKILL.md` などのファイルにタスク手順・知識・制約を記述することで、エージェントが特定の状況で自動的にまたは明示的に呼び出せる「スキル」を追加できる。

## 詳細

### Agent Skills オープンスタンダード

[agentskills.io](https://agentskills.io) が策定するオープンスタンダードで、Claude Code・OpenAI Codex CLI など複数のツールが採用している。`SKILL.md` ファイルに YAML フロントマターと Markdown 形式の指示を記述するだけで、ツール横断のスキル共有が可能になる。

### Claude Code におけるスキル

Claude Code ではスキルが `/slash-command` として機能し、個人・プロジェクト・エンタープライズの3スコープで管理できる。

#### 格納場所

| 場所 | パス | 対象範囲 |
|------|------|---------|
| エンタープライズ | 管理設定経由 | 組織全体 |
| 個人 | `~/.claude/skills/<skill-name>/SKILL.md` | 全プロジェクト |
| プロジェクト | `.claude/skills/<skill-name>/SKILL.md` | 当該プロジェクトのみ |
| プラグイン | `<plugin>/skills/<skill-name>/SKILL.md` | プラグイン有効化プロジェクト |

#### SKILL.md の構造

```yaml
---
name: スキル名
description: スキルの説明（Claude がいつ使うか判断する）
disable-model-invocation: true  # ユーザーのみ呼び出し可能にする
context: fork                    # サブエージェントとして実行
agent: Explore
allowed-tools: Read, Grep
---

スキルの指示内容（Markdown）
```

#### 主要フロントマターフィールド

- `name`: スラッシュコマンド名（例: `commit` → `/commit`）
- `description`: Claude が自動呼び出しを判断するための説明
- `disable-model-invocation`: `true` で Claude の自動呼び出しを禁止（デプロイ等の副作用があるスキル向け）
- `user-invocable`: `false` でスラッシュコマンドメニューから非表示にし Claude のみが使用
- `context: fork`: サブエージェントとして独立コンテキストで実行
- `allowed-tools`: スキル実行中に許可するツールの一覧

#### 動的コンテキスト注入

`` !`command` `` 構文でシェルコマンドの出力をスキル内容に埋め込める。Claude にデータを渡す前処理として活用できる。

```yaml
---
name: pr-summary
context: fork
agent: Explore
---

PR の差分: !`gh pr diff`
PR のコメント: !`gh pr view --comments`
```

### 他 AI ツールのスキル実装

| ツール | 実装 | 特徴 |
|--------|------|------|
| Claude Code | `SKILL.md` (Agent Skills 標準) | スコープ管理・サブエージェント連携・動的コンテキスト注入 |
| OpenAI Codex CLI | `skills.md` (Agent Skills 標準) | 基本的な Agent Skills 標準に準拠 |
| Cursor | `.cursor/rules/`（カスタムルール） | ファイルパターンや状況に応じてルールを自動適用 |
| Windsurf | Cascade ルール | ファイル・ターミナル・クリップボード履歴を追跡して次の操作を予測 |

### スキルとサブエージェントの違い

| 観点 | スキル | サブエージェント |
|------|--------|----------------|
| 実行コンテキスト | メイン会話内（デフォルト） | 独立したコンテキストウィンドウ |
| 呼び出し方 | スラッシュコマンド / Claude 自動 | Claude が Task ツールで委任 |
| 定義ファイル | `SKILL.md` | `agents/*.md` |
| 向いている用途 | 再利用可能なプロンプト・知識 | 独立した複雑タスク・ツール制限 |

## ポイント

- スキルはプロジェクトの `.claude/skills/` にコミットすることでチーム共有できる
- `disable-model-invocation: true` を設定することでデプロイやコミットなど副作用のある操作を誤って自動実行させない
- `context: fork` で指定したサブエージェント内でスキルを実行するとメイン会話のコンテキストを汚染しない
- スキル説明文は Claude が自動呼び出しするかどうかを決定する重要な要素で、ユーザーが自然に使いそうなキーワードを含めると良い
- Agent Skills オープンスタンダードにより複数ツール間でスキルを共有・移植できる可能性がある
