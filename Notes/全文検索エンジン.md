## 概要

全文検索エンジン（Full-Text Search Engine）は，大量のテキストドキュメントを高速に検索するために転置インデックスを用いるデータストアである．Elasticsearch・OpenSearch がデファクトスタンダードで，商品検索・ログ解析・ナレッジ検索など，テキストの柔軟な検索・スコアリング・集計が必要なシステムで広く採用される．

## 詳細

### 転置インデックスの仕組み

転置インデックスは「単語 → ドキュメント ID のリスト」というマッピングを保持する：

```
通常の格納（ドキュメント → 単語）:
  doc1: "機械学習 は AI の 基礎"
  doc2: "深層学習 は 機械学習 の 一種"

転置インデックス（単語 → ドキュメント）:
  機械学習 → [doc1, doc2]
  AI       → [doc1]
  深層学習  → [doc2]
  基礎     → [doc1]
  一種     → [doc2]
```

検索クエリ「機械学習」に対して，インデックスを引くだけで即座に対象ドキュメントを返せる．

### テキスト解析パイプライン

```
入力テキスト
    ↓ Tokenizer（分割）
   ["機械", "学習", "は", "AI"]
    ↓ Token Filter（正規化・除去）
   ["機械", "学習", "AI"]  ← ストップワード除去，ステミング等
    ↓ 転置インデックスへ格納
```

日本語では Kuromoji（形態素解析）または n-gram が主に使われる．

### Elasticsearch の主要機能

| 機能 | 説明 |
|------|------|
| **全文検索** | BM25 スコアリングによる関連度順の検索 |
| **ファセット（集計）** | カテゴリ別カウント，数値レンジ集計 |
| **ハイライト** | 検索キーワードのスニペット表示 |
| **オートコンプリート** | Completion Suggester でサジェスト |
| **地理空間検索** | Geo-point/Geo-shape による近傍検索 |
| **ベクトル検索** | Dense vector フィールドで ANN（近似最近傍）検索 |
| **EQL（Elastic Query Language）** | セキュリティ・ログ向けのイベントクエリ |

### 主要製品

| 製品 | 特徴 |
|------|------|
| **Elasticsearch** | 最も普及した全文検索エンジン．Elastic スタック（ELK）の中核 |
| **OpenSearch** | AWS が Fork した Elasticsearch の OSS 版．完全互換 |
| **Apache Solr** | Apache Lucene ベース．エンタープライズ向け実績が豊富 |
| **Meilisearch** | Rust 製の高速軽量全文検索（組み込み・セルフホスト向け） |
| **Typesense** | シンプルな API の高速全文検索（Meilisearch の競合） |

### Elastic スタック（ELK）

Elasticsearch を中心とした観測性・ログ分析スタック：

```
ログ・メトリクス源
    ↓
Logstash / Beats（収集・変換）
    ↓
Elasticsearch（格納・インデックス）
    ↓
Kibana（可視化・ダッシュボード）
```

### ユースケース

- **EC 商品検索**: キーワード・カテゴリ・価格レンジでの商品検索・ファセット
- **ログ解析**: アプリケーションログの全文検索・異常検知（SIEM 用途）
- **社内ナレッジ検索**: ドキュメント・Wiki・Issue の横断検索
- **オートコンプリート**: 検索ボックスのリアルタイムサジェスト
- **セキュリティ分析**: SIEM（Elastic Security）でのサイバー攻撃検知

## ポイント

- 転置インデックスにより「LIKE 検索」より桁違いに速いテキスト検索を実現
- Elasticsearch は検索専用DBではなくログ集約・分析基盤としても広く使われる
- [[ポリグロット永続化]] では検索特化のデータストアとして RDBMS と並走させるパターンが多い
- 2026年トレンド：ベクトル検索との統合（ハイブリッド検索）が加速，Elasticsearch 9.0 がネイティブ対応
- OpenSearch は AWS エコシステムで Elasticsearch の代替として急速に普及
