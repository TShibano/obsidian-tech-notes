## 概要

HedgeDoc（旧称 CodiMD）はリアルタイム共同編集に対応したオープンソースの Markdown ノートプラットフォーム。ブラウザ上でリアルタイムにチームメンバーと同一ドキュメントを編集でき、セルフホスト可能なことからプライバシーやデータ主権を重視する組織に人気がある。

### 主要機能

| 機能 | 説明 |
|------|------|
| リアルタイム共同編集 | 複数ユーザーが同時に同一文書を編集、変更が即時反映 |
| Markdown サポート | 標準 Markdown + 拡張構文（コードブロック・数式・図表） |
| スライド作成 | Reveal.js 連携でプレゼンテーションをその場で生成 |
| バージョン履歴 | 自動保存・差分確認・以前の状態への復元 |
| エクスポート | Markdown・HTML・PDF 形式でエクスポート可能 |
| 権限管理 | 閲覧・コメント・編集の権限をドキュメント単位で設定 |
| ゲスト共有 | URLを共有するだけでゲストユーザーが閲覧・編集可能 |

### CodiMD からの経緯

元々 HackMD（商用）のオープンソース版として CodiMD の名称で開発されていた。コミュニティフォークとして独立した後、プロジェクトを **HedgeDoc** に改名（2020年）。現在は `hedgedoc/hedgedoc` を公式リポジトリとして独立した組織が開発を継続している。

### ユースケース

- チーム内ドキュメントの共同作成・レビュー
- ミーティングメモのリアルタイム同時記録
- 技術仕様・設計書の共同編集
- オフラインイベント・ハッカソンでの情報共有
- 自己ホストによるプライベートな Wiki

### 他ツールとの比較

| ツール | 特徴 | 向き不向き |
|--------|------|-----------|
| **HedgeDoc** | セルフホスト可・Markdownファースト・軽量 | エンジニア向け共同ドキュメント |
| **[[Notion]]** | 多機能・クラウド専用・データベース統合 | 非技術者を含むチーム全体 |
| **[[Outline]]** | チーム Wiki に特化・整理しやすい構造 | ナレッジベース構築 |
| **[[Obsidian]]** | ローカル完結・グラフ表示・双方向リンク | 個人の知識管理 |

## ポイント

- セルフホスト運用によりデータをサーバーに置かずに済む（GDPR/社内コンプライアンス対応）
- AGPL 3.0 ライセンスで商用利用も可能（ただし派生物のソース公開義務あり）
- HackMD の商用機能（プライベートノート・組織管理など）が必要な場合は [HackMD](https://hackmd.io) が適切
- スライドや数式表示も標準対応しており、学術・技術文書との相性が良い
