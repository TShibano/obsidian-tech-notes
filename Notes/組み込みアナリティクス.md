## 概要

BI・レポーティング・データ可視化などのアナリティクス機能を，別ツールに切り替えることなくアプリケーション内にシームレスに統合する手法．エンドユーザが使い慣れたアプリケーションの中でデータ分析を行える環境を提供する．

## 詳細

### 定義

アナリティクス機能（ダッシュボード，グラフ，レポート，データ可視化）をソフトウェアアプリケーションやポータルサイトに直接組み込むアプローチ．ユーザは分析ツールに切り替えることなく，業務システムの中でデータ洞察を得られる．

### 通常の BI との違い

| | 外部 BI ツール | 組み込みアナリティクス |
|---|---|---|
| アクセス方法 | 専用ツールに切り替える | アプリ内で直接利用 |
| ユーザ体験 | コンテキストスイッチが発生 | シームレスな体験 |
| 導入対象 | データアナリスト中心 | アプリのすべてのユーザ |
| カスタマイズ | ツールに依存 | アプリに合わせて完全カスタマイズ |

### 主な構成要素

- **Analytics Platform as a Service (AnPaaS)**: 分析エンジンのクラウドサービス
- **データ接続層**: 複数データソースを統合する統一データレイヤ
- **データ可視化**: チャート・グラフ・ダッシュボードのコンポーネント
- **予測分析**: 統計モデル・機械学習による将来予測機能
- **API / SDK**: アプリケーションへの統合インターフェース

### 統合方式

1. **iFrame 埋め込み**: BI ツールの画面を iFrame で表示（手軽だがカスタマイズ制限あり）
2. **API 統合**: 分析エンジンの API を呼び出してデータを取得し，自前の UI で表示
3. **SDK 統合**: ベンダー提供の SDK でコンポーネントを組み込む
4. **セルフビルド**: 自社で分析エンジンを構築（工数大，柔軟性高）

### ビジネス価値

- 未活用データを価値ある資産に変換
- 複数の視点からのデータ分析をユーザに提供
- 複雑な技術実装を簡素化しながら開発コストを削減
- データ活用の民主化（技術者以外もデータを活用可能）

### 主要プロダクト

- **Yellowfin** : OEM 対応の組み込み BI
- **Qlik Embedded Analytics**: 高度な埋め込み分析
- **Salesforce Analytics**: CRM への組み込み分析
- **Oracle Embedded Analytics**: エンタープライズ向け

## ポイント

- SaaS プロダクトへの競合優位性として「分析機能」の組み込みが差別化要因になっている
- 「構築するか購入するか（Build or Buy）」の意思決定が重要
- [[オペレーショナルアナリティクス]] と組み合わせることでリアルタイムな組み込み分析が実現
