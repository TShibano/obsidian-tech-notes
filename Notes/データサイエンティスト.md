## 概要

データサイエンティスト（Data Scientist）は，統計学・機械学習・プログラミングを駆使して，大量のデータから予測モデルの構築・仮説検証・複雑なパターン発見を行う役割である．ビジネス課題に対して，機械学習やAIを用いた高度な分析・モデリングでソリューションを提供し，データサイエンスの研究開発を担当することもある．

## 詳細

### データアナリストとの違い

| 観点 | データサイエンティスト | [[データアナリスト]] |
|------|---------------------|---------------|
| 主な目的 | 予測モデル構築，複雑なパターン発見 | ビジネス状況の理解・可視化 |
| 分析手法 | 機械学習，深層学習，高度な統計 | 記述統計，集計，トレンド分析 |
| 主なツール | Python（scikit-learn，PyTorch，TF），R | SQL，BI ツール，Excel |
| 機械学習 | 設計・実装・評価 | 基礎知識（使う程度） |
| 数学・統計 | 高い（線形代数，確率論等） | 基礎〜中程度 |
| プログラミング | 高い（モデル実装） | SQL 中心，Python は補助的 |

### 主な職務

| 業務 | 内容 |
|------|------|
| **課題設定** | ビジネス課題をデータサイエンス問題として定式化 |
| **データ探索（EDA）** | 探索的データ分析でデータの特性・パターンを把握 |
| **特徴量エンジニアリング** | モデルの入力となる変数の設計・変換 |
| **モデリング** | 機械学習モデルの構築・学習・評価（CV，ハイパーパラメータ調整） |
| **デプロイ・モニタリング** | モデルの本番導入と性能モニタリング |
| **研究・論文調査** | 最新の ML/AI 手法の調査・適用検討 |

### 必要なスキルセット

- **統計・数学**: 確率論，線形代数，最適化，ベイズ統計
- **機械学習**: 教師あり学習（回帰・分類），教師なし学習，深層学習
- **プログラミング**: Python（必須），R（統計寄り），SQL
- **ライブラリ**: scikit-learn，PyTorch，TensorFlow，Keras，pandas，NumPy
- **MLOps**: モデル管理（MLflow），特徴量ストア，CI/CD
- **ドメイン知識**: 対象ビジネス領域の理解

### データエンジニアとの協働

[[データエンジニア]] がデータ基盤・パイプラインを構築し，データサイエンティストがそのデータを使ってモデルを開発する．モデルの本番化（MLOps）では両者が密に連携する．

## ポイント

- [[データアナリスト]] が「過去・現在を説明する」のに対し，データサイエンティストは「未来を予測する」
- 統計・機械学習・ドメイン知識の3つが揃って初めて価値を発揮できる「T字型スキル」が理想
- 2026年のトレンド：LLM の活用，AutoML の普及，MLOps の標準化
- コードを書けるだけでなく，ビジネス課題を適切に定式化できることが差別化要因
